# IronLedger Gym Website - Project Overview

**Project Name:** IronLedger

**Project Type:** Gym & Fitness Website

## Overview

IronLedger is a modern fitness and gym website designed to help users learn about gym services, fitness programs, trainers, and membership plans. The website provides an attractive interface where visitors can explore workout programs, view trainer details, and contact the gym for enrollment.

The main goal of the IronLedger website is to promote a healthy lifestyle and provide information about fitness training, strength building, weight loss programs, and gym memberships.

## Features

✅ Responsive Homepage

✅ About Us Section

✅ Fitness Programs Display

✅ Trainer Information

✅ Membership Plans

✅ Gallery Section

✅ Contact Form

✅ Modern Navigation Bar

✅ Footer with Social Media Links

## Technologies Used

* HTML5
* CSS3
* JavaScript (Optional)
* Google Fonts
* Font Awesome Icons

## Modules

### 1. Home Page

Displays the gym's introduction, motivational slogan, and call-to-action buttons.

### 2. About Us

Provides information about IronLedger's mission, vision, and fitness philosophy.

### 3. Programs Section

Shows available fitness programs such as:

* Weight Training
* Cardio Training
* Personal Training
* Strength Building
* Body Transformation

### 4. Trainers Section

Displays professional trainers and their expertise.

### 5. Membership Plans

Shows pricing and benefits of different membership packages.

### 6. Contact Section

Allows users to send inquiries and find gym contact details.

## Project Objective

The objective of IronLedger is to create a professional and user-friendly gym website that attracts potential members and provides complete information about fitness services offered by the gym.

## Conclusion

IronLedger is a simple yet effective fitness website developed using HTML and CSS. It demonstrates web design skills, responsive layouts, and modern UI principles while providing a platform for promoting gym and fitness services.
